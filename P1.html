The final output of this program is not a financial advice.

This program selects the 50 stocks with the highest price momentum from the S&P500 universe. Then, the program will take a numeric input from the user, that will be used as the total capital to invest in the portfolio, and recommend the number of stocks to buy from each company to have an equal weight portfolio.

The program takes a .csv file that contains the 500 companies that are in the S&P500. Using the IEX cloud API, the program will request the data for each company, such as price, one year price return, six months price return, three months price return, and one month price return. From there, the program will score each stock based on the mean of the percentiles of the time frames, this metric is called HQM (High Quality Momentum) score. The program will filter the top 50 stocks with the highest HQM score.

The program can take any .csv input, as far as the ticker symbols in it are public companies. The reason to use the companies in the S&P500 is because these are the biggest companies in terms of market capitalization, which implies certain stability in their prices and thus mitigates certain risk factors. The S&P500 is also diversied among the most important economic sectors, which also helps to mitigate certain risk factors. The program outputs a excel file with the final results.
