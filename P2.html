Compilation of different data visualization tasks that I performed for work and school projects.

The data used for these visuals were retrieved from the following sites:

  • https://www.census.gov/quickfacts/fact/table/US/PST045219
  • https://www.bea.gov/data
  • https://github.com/CSSEGISandData/COVID-19
  • https://www.bls.gov/data/tools.htm

The tools used to create the visuals are Tableau, Python (Plotly and Dash), and Excel. Some of them retrieved the data from a data base connected through an API, others had to take the data directly from a .csv file, or would download the .csv file from the corresponding website and then use it in the program.
