The final output of this program is not a financial advice.

This program is very similar to "Qualitative Momentum Strategy" shown in project 1. The same as before, I used a .csv file to specify the ticker symbol of a given company to the program. I also used the IEX cloud IPA to obtain the data of the companies. However, in certain cases the API would not have some metrics needed for this program, in such cases I had to compute the metric within the program.

Instead of searching for companies that have a positive momentum in their prices, this program aim to identify the companies that have the best value metrics. These value metrics are:

  • Price to earning ratio
  • Price to book ratio
  • Price to sales ratio
  • EV/EBITDA
  • EV/GP

From these metrics, the program will score each stock based on the mean of the percentiles of the metrics, the final score is called RV (Robust Value) score. The program will filter the top 50 stocks with the highest RV score.

The program can take any .csv input, as far as the ticker symbols in it are public companies. The reason to use the companies in the S&P500 is because these are the biggest companies in terms of market capitalization, which implies certain stability in their prices and thus mitigates certain risk factors. The S&P500 is also diversied among the most important economic sectors, which also helps to mitigate certain risk factors. The program outputs a excel file with the final results.
